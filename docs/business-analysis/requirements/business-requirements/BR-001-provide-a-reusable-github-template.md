## Summary

ForgePilot shall be delivered as a reusable GitHub template repository that can be used to instantiate new repositories or be applied to existing ones.

## Details

This requirement defines the delivery form of ForgePilot. The system must provide a ready-to-use repository that includes the required directory structure, scripts, templates, examples, and documentation. The template should serve as a starting point for any new project and should be adaptable for integration into an existing repository without losing existing content or breaking conventions.

## Acceptance Criteria

- A GitHub template repository exists with the specified structure.
- The template can be used to create new repositories via GitHub's "Use this template" feature.
- The template can be applied to existing repositories without destructive changes.
- All required directories and files are present.

## Notes

This requirement establishes the core deliverable for ForgePilot.
