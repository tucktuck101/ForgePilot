## Description

Provide guidance and templates but leave discovery, issue creation, delivery orchestration, validation, and handoff coordination to the user.

## Analysis

- Benefits: simplest implementation and lowest automation risk.
- Costs: high manual coordination and weak support for the autonomy goal.
- Risks: inconsistent process, missed gates, and incomplete traceability.
- Conclusion: retain as a fallback operating mode, not the primary direction.
