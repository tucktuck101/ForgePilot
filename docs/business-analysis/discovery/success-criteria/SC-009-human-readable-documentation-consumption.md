## Criterion

A representative user, reviewer, or maintainer can understand the project state, approval state, and major artefact relationships through readable summaries, indexes, and linked detail.

## Measurement

Run a task-based review using the repository documentation or a later approved presentation layer. Target: the participant locates the requested overview and authoritative detail without needing Codex to reproduce the documentation.
