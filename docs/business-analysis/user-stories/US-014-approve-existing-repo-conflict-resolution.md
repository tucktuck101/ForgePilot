## Summary

Capture the Client need to approve conflict resolutions before existing files change.

## Story

As the Client, I want to approve conflict resolutions before ForgePilot changes existing files so that valuable repo content is not lost.

## Acceptance Criteria

- Conflicts are listed with recommended resolutions.
- Alternatives are provided.
- Codex waits for approval before resolving conflicts.
