## Story

As Codex, I want to create GitHub Issues from the approved scope and solution design so that delivery work is executable and traceable.

## Acceptance Criteria

- Issues map to scoped work.
- Issues link to relevant docs.
- Issues include acceptance criteria and test expectations.
- Issues are ordered by dependency and value.
