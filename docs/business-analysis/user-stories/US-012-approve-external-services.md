## Summary

Capture the Client need to approve external services before Codex adds them.

## Story

As the Client, I want to approve external services before Codex adds them so that cost, account, privacy, and vendor-lock-in risks are controlled.

## Acceptance Criteria

- Service name and purpose are identified.
- Cost/security/privacy implications are described.
- Free/local/mock alternatives are considered.
- Codex waits for approval before use.
