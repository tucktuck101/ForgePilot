## Summary

Capture the Solo Builder need for chat-based discovery.

## Story

As a Solo Builder, I want Codex to interview me through chat so that my idea can be converted into project artefacts.

## Acceptance Criteria

- Codex uses CLI chat or Codex chat window.
- Codex asks one focused question at a time.
- Answers are captured in discovery artefacts.
- Codex stops once enough context exists for product brief and scope approval.
