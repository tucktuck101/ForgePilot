## Summary

Capture the Codex need for reusable delivery skills.

## Story

As Codex, I want reusable skills so that delivery workflows are consistent and token-efficient.

## Acceptance Criteria

- Skills live under `.agents/skills/`.
- Each skill has a clear trigger, process, outputs, stop conditions, and approval gates.
- Skills avoid duplicating large instructions.
