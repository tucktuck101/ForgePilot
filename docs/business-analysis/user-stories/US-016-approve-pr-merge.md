## Summary

Capture the Client need to approve pull request merges.

## Story

As the Client, I want to approve PR merges so that no sprint is merged without human review.

## Acceptance Criteria

- PR links to its issue and sprint notes.
- Tests/checks are recorded.
- Risks and approval gates are listed.
- Codex does not merge without approval.
