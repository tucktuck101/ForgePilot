## Summary

Capture the human need to navigate and understand ForgePilot documentation without creating a separate source of truth.

## Story

As a user, reviewer, or maintainer, I want a readable, navigable, and searchable documentation path so that I can understand ForgePilot and follow related detail from the authoritative repository Markdown.

## Acceptance Criteria

- A reader can start from a human-facing summary or index and locate major artefacts.
- Related detail is available through links rather than duplicated narrative.
- The current approval state and provisional decisions are visible.
- Any later portal presents content derived from repository Markdown and does not become a competing authoritative store.
- Portal tooling, site design, and publication mechanics remain subject to later approved design and implementation work.
