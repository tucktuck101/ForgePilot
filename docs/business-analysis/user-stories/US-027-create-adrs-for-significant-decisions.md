## Summary

Capture the Codex need to record significant decisions without creating documentation noise.

## Story

As Codex, I want to create ADRs only for significant decisions so that important trade-offs are traceable without creating documentation noise.

## Acceptance Criteria

- ADRs are created for qualifying decisions.
- Routine choices go in sprint notes.
- ADRs include context, decision, consequences, alternatives, and links.
