## Summary

Capture the Solo Builder need to start a new project from the ForgePilot template.

## Story

As a Solo Builder, I want to create a new GitHub repo from the ForgePilot template so that I can start prototyping a software idea quickly.

## Acceptance Criteria

- The template can be used directly as a GitHub template repo.
- Required files and folders are present after repo creation.
- First-run instructions are available.
- Codex can begin repo intake from the initial state.
